# 252104 尋找低谷

```python
heights = list(map(int, input().split()))  # 山陵線的數值

n = len(heights)  # 山稜線數值列表長度

valley_index = -1  # 先放一個不可能的列表位置值

for i in range(1, n - 1):
    if heights[i] < heights[i - 1] and heights[i] < heights[i + 1]:  # 低谷就是比左邊低且比右邊低
        valley_index = i
        break  # 題目保證只有一個低谷，找到就可以跳出

print(valley_index)  # 低谷出現的位置
```
