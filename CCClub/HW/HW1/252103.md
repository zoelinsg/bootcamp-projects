# 252103 出題目

```python
a = int(input())
b = int(input())
c = int(input())  # x + y = c
d = int(input())  # a*x + b*y = d

# ① a == b 的情況
if a == b:
    print("False")          # 無解和無限多解
else:
    numerator = d - b * c      # d - b*c 分子
    denominator = a - b        # a - b 分母

    # x 不是整數的情況
    if numerator % denominator != 0:
        print("False")      # 這裡是「有實數解，但 x 不是整數」→ 不合格
    else:
        x = numerator // denominator
        y = c - x

        # x 或 y 不是正整數<= 0的情況
        if x > 0 and y > 0:
            print("True")   # 唯一正整數解
        else:
            print("False")  # 這裡是解是 0 或負數
```
