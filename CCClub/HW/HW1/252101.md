# 252101 凱凱的出門準備

```python
H, L = map(int, input().split())
P = int(input())
U = int(input())

items = []  # 要帶的物品列表

# 雨傘：雨天 或 UV >= 6
if P >= 70 or U >= 6:
    items.append("雨傘")

# 帽子：先判斷毛帽，再判斷棒球帽（毛帽優先）
avg = (H + L) / 2
if avg <= 18:
    items.append("毛帽")
elif (20 < P and P < 70) and L <= 20:
    items.append("棒球帽")

# 太陽眼鏡：晴天 且 UV >= 3
if P <= 20 and U >= 3:
    items.append("太陽眼鏡")

# 輸出
if len(items) == 0:
    print("空手出門")
else:
    print(" ".join(items))
```
