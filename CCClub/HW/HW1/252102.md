# 252102 綜合1-4：括號合法

```python
s = input().strip()

balance = 0
is_valid = True

for ch in s:
    if ch == '(':
        balance += 1
    elif ch == ')':
        if balance == 0:
            is_valid = False
            break
        else:
            balance -= 1
    else:
        is_valid = False
        break

if balance != 0:
    is_valid = False

if is_valid:
    print("True")
else:
    print("False")
```
